# 📊 FinIntel Pro: Comprehensive AI Auditor
FinIntel Pro is a Generative AI-powered financial analysis tool designed to automate the auditing of complex financial statements (10-K, 10-Q). By leveraging Llama 3.1 and Groq's LPU technology, it turns static PDFs into interactive, actionable insights in seconds.

# 🚀 Key Features
Automated Financial Extraction: Instantly identifies Net Income, Revenue, and Cash Flow from uploaded PDFs.

Interactive AI Auditor: A dedicated chat interface to query specific risks, debt structures, or growth drivers within the document.

Blazing Fast Inference: Integrated with Groq for sub-second response times using the llama-3.1-8b-instant model.

Professional UI: Custom-styled Streamlit interface for a premium user experience.

# 🛠️ Tech Stack
LLM: Llama 3.1 8B (via Groq API)

Framework: Streamlit

Document Processing: PyPDF & LangChain

Environment Management: Python-dotenv
